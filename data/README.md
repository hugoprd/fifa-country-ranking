# Extração dos Dados

A base de dados utilizada para o projeto inteiro foi, principalmente, da FBref que contém dados de 1930 até 2026.

## Competições Continentais

### 1. América do Sul (CONMEBOL)
    - Copa Libertadores: https://fbref.com/en/comps/14/
    - Copa Sudamericana: https://fbref.com/en/comps/205/

### 2. Europa (UEFA)
    - UEFA Champions League: https://fbref.com/en/comps/8/
    - UEFA Europa League: https://fbref.com/en/comps/19/
    - UEFA Europa Conference League: https://fbref.com/en/comps/882/

### 3. América do Norte, Central e Caribe (CONCACAF)
    - CONCACAF Champions Cup: https://fbref.com/en/comps/133/

### 4. Ásia (AFC)
    - AFC Champions League: https://fbref.com/en/comps/664/

## Competições Mundiais

### 1. Mundial de Clubes
    - https://fbref.com/en/comps/719/

### 2. Copa do Mundo
    - https://fbref.com/en/comps/1/

# Uso dos dados

O objetivo foi pegar os dados de todas as competições mundiais para ser possível pedir a capacidade de cada liga do planeta.
*( Infelizmente, a competição do continente África (CAF) não possuía dados disponíveis de fácil acesso, então foi necessário pegar dados de times africanos através do `Mundial de Clubes` )*
