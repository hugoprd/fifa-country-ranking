# Fonte dos Dados

A base de dados utilizada para o projeto foi do FootyStats para as competições globais, do próprio Wikipedia para competições continentais e do Kaggle "Football Data from Transfermarkt" para os jogadores.

## Competições Continentais

Foi feito um webscrapping em cada página da Wikipedia de cada competição continental para a obtenção dos nomes dos times, suas confederações e seus países.

1. CAF: https://en.wikipedia.org/wiki/List_of_top-division_football_clubs_in_CAF_countries
2. CONMEBOL: https://en.wikipedia.org/wiki/List_of_top-division_football_clubs_in_CONMEBOL_countries
3. UEFA: https://en.wikipedia.org/wiki/List_of_top-division_football_clubs_in_UEFA_countries
4. CONCACAF: https://en.wikipedia.org/wiki/List_of_top-division_football_clubs_in_CONCACAF_countries
5. AFC: https://en.wikipedia.org/wiki/List_of_top-division_football_clubs_in_AFC_countries
